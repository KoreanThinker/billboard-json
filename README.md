# Usage
### Curl
```bash
$ curl https://raw.githubusercontent.com/KoreanThinker/billboard-json/main/billboard-hot-100/recent.json
```
### fetch
```js
fetch('https://raw.githubusercontent.com/KoreanThinker/billboard-json/main/billboard-hot-100/recent.json')
  .then(res => res.json())
  .then(data => console.log(data))
```

# Type
### Song/Album Charts (`Billboard Hot 100`, `Billboard 200`, `Billboard Global 200`, and most other charts)
```json
{
  "date": "YYYY-DD-MM",
  "data": [
    {
      "name": "string",
      "artist": "string",
      "image": "string",
      "rank": "number",
      "last_week_rank": "number | null",
      "peak_rank": "number",
      "weeks_on_chart": "number"
    },
    "..."
  ]
}
```
### Artist/Songwriter/Producer Charts (`Artist 100`, `Hot 100 Songwriters`, `Hot 100 Producers`, etc.)
```json
{
  "date": "YYYY-DD-MM",
  "data": [
    {
      "name": "string",
      "image": "string",
      "rank": "number",
      "last_week_rank": "number | null",
      "peak_rank": "number",
      "weeks_on_chart": "number"

}
```
